How are networks ranked?

Networks are ranked based on their retrieval score—the higher the score, the higher the position on the leaderboard.

How is retrieval score determined for Filecoin?

Filecoin retrieval score represents average of all Storage Provider (SP) retrieval scores (RSR) over 24-hour period. Storage Provider RSR is calculated by dividing the number of successful retrievals by the total valid retrieval requests for that provider within a given timeframe. Only requests that pass fraud detection are included in the calculation. For more detail please refer to Spark protocol docs.

How is retrieval score determined for Arweave?

The retrieval score for Arweave is determined by the percentage of successful retrievals over a 24-hour period. Retrievals are conducted by selecting a random Arweave transaction and performing a retrieval from a random Arweave node, and the results are reported back to the API (without fraud checking). For more details, refer to the Arweave Checker repository.

How is retrieval score determined for Walrus?

The retrieval score for Walrus is calculated based on the percentage of successful retrievals over a 24-hour period. Retrievals on Walrus are performed by selecting a random blob and fetching it from one of the public aggregator nodes and result of a retrieval check is reported back to the API (without fraud checking). Note that this approach currently results in a higher retrieval score, as it does not involve decentralized node retrievals (unlinke Filecoin and Arweave). For more details, refer to the Walrus Checker repository.
